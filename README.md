# Neon Ball Sort Pro 🧪✨

A premium, neon-styled puzzle game developed by **AliApps1**. Challenge your brain by sorting colored balls into the correct tubes.

## Features 🚀
* **Multilingual Support:** English (Default), Arabic, and Persian.
* **Neon Aesthetics:** High-quality glowing visuals and smooth animations.
* **User Preferences:** Customizable Sound, Vibration, and Auto-move settings.
* **Cross-Platform:** Built with HTML5, CSS3, and JavaScript for mobile and web.

## Project Structure 📂
* `index.html`: Main game structure.
* `style.css`: Neon styling and RTL support.
* `script.js`: Game logic and translation system.
* `privacy-policy.html`: Official privacy documentation.

## How to Play 🎮
1. Tap any tube to select the top ball.
2. Tap another tube to move the ball.
3. You can only move a ball onto a ball of the same color or into an empty tube.
